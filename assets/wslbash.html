Windows Subsystem for Linux is a new feature offered by Windows and was released from Windows version 8.1, WSL received its official stable release from Windows 10.

WSL provides a Linux compatible kernel interface which is developed by Microsoft. It provides Linux kernel which allow user to run the GNU user space such as Ubuntu. It enables alongside bash scripting with Linux kernel and Windows GUI rather than Windows NT kernel.

WSL provides users to select the GNU, utilities, command-line tools and application on Windows.

2.   System Admin

3.   Linux GUI

4.   Multiple command-line tools and utilities

 

-1.2- Why WSL?

Okay so when it comes to a kernel installation alongside windows most users are aware about the Virtual Box (VB). Virtual Box is an application use to run Ubuntu, Parrot OS, Linux and many other OS alongside Windows, a stable GNU for most of the users.

Where does the VB fails?

Well a Virtual Box enables the user to access the content of user PC along side window but wont allow the user to perform GUI windows and Kernel Linux operations. Virtual Box will take a small storage of your PC and install a new OS which you can operate while using Windows as main OS and VM with Ubuntu, Kali Linux, Parrot OS, Sun sys etc as a substitute OS.

GUI and Kernel of the same OS for VB

WSL is completely opposite to VM, here the user uses Windows GUI while Linux Kernel in a stable form. Isnt it amazing Windows GUI + Linux terminal/Kernel with a faster performance without even giving a complete installation to GUI of other OS. You can install any command-line tool, system utilities and Kernel based applications that are to be performed.

-1.3- WSL-1 vs WSL-2

WSL 2 is a new version of the architecture that powers the Windows Subsystem for Linux to run ELF64 Linux binaries on Windows. Its primary goals are to increase file system performance, as well as adding full system call compatibility. This new architecture changes how these Linux binaries interact with Windows and your computers hardware, but still provides the same user experience as in WSL 1 (the current widely available version). Individual Linux distros can be run either as a WSL 1 distro, or as a WSL 2 distro, can be upgraded or downgraded at any time, and you can run WSL 1 and WSL 2 distros side by side. WSL 2 uses an entirely new architecture that uses a real Linux kernel. Its a major reworking of the original WSL concept, moving away from translating Linux system calls to Windows to shipping a complete Linux kernel that runs alongside Windows own kernel.

WSL system performance is 20x faster as compared to WSL 1. Initial versions of WSL 2 run up to 20x faster compared to WSL 1 when unpacking a zipped tarball, and around 2-5x faster when using git clone, npm install and cmake on various projects.

Loops in bash

Loops allow us to take a series of commands and keep re-running them until a particular situation is reached. They are useful for automating repetitive tasks.

There are 3 basic loop structures in Bash scripting which we'll look at below. There are also a few statements which we can use to control the loops operation.

While loop

Syntax:

while [ <some test> ]
do
<commands>
done

Sample program:

1.    counter=1

2.    while [ $counter -le 10 ]

3.    do

4.    echo $counter

5.    ((counter++))

6.    done

7.     

8.    echo All done

until [ <some test> ]
do
<commands>
done

5.    ((counter++))

Bash Array

Bash supports one-dimensional numerically indexed and associative arrays types. Numerical arrays are referenced using integers, and associative are referenced using strings.

Numerically indexed arrays can be accessed from the end using negative indices, the index of -1 references the last element. The indices do not have to be contiguous.

Unlike most of the programming languages, Bash array elements dont have to be of the same data type. You can create an array that contains both strings and numbers.

Bash does not support multidimensional arrays, and you cant have array elements that are also arrays.

There is no limit on the maximum number of elements that can be stored in an array.

Creating Array:

Indexed array

declare -a array_name

Method 1 for decleration

array_name[index2]=value2

Features of Vim:

      It supports recording features which allows to record and play Vim commands in repeated manner

Moving with Vim (navigation)

Editing
